# docs/journal

Purpose: chronological design and execution trace.

Belongs:

- dated decision notes
- handoff records
- retrospectives
- rationale snapshots
- historical traceability

Not allowed:

- current work ledger ownership
- checklist authority
- finalized specification text

Journal entries may explain why something happened, but they do not become
normative merely by existing here.
